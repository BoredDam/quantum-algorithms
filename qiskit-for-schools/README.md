# Notebook giornata delle scuole Qiskit Fall Fest 2025 

Questo notebook nasce come introduzione al Quantum Computing per i ragazzi delle scuole superiori coinvolte al Qiskit Fall Fest 2025, al DMI dell'Università di Catania.

## Per eseguire questo notebook

Crea una cella di codice con le seguenti righe ed eseguila, oppure esegui il comando da terminale per usare il notebook in locale, per scaricare gli opportuni pacchetti Qiskit.

Da terminale (è prima consigliabile essere in un virtual environment per non scassare tutto! Allego una guida, [clicca qui](https://github.com/BoredDam/notes-to-self-and-utils/tree/main/about-VENVs)):
```bash
pip install qiskit
pip install qiskit-ibm-runtime
pip install qiskit[visualization]
pip install qiskit-aer
```

o sul notebook online:
```bash
%pip install qiskit
%pip install qiskit-ibm-runtime
%pip install qiskit[visualization]
%pip install qiskit-aer
```
