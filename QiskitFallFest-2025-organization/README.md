## Workshop #2, Criteri di DiVincenzo

I criteri di DiVincenzo (dal fisico David DiVincenzo) definiscono i cinque requisiti per definire un sistema fisico un computer quantistico.

1. Un sistema fisico scalabile con qubit ben definiti.
2. Inizializzazione dei qubit a valori arbitrari.
3. Tempi di decoerenza sufficienti.
4. Un set universale di gate.
5. Possibilità di misurare specifici qubit.

Se questi cinque criteri sono rispettati dal sistema, allora si può effettuare computazione quantistica.
Con il Notebook proposto si verificano questi requisiti e si mettono in pratica degli esperimenti.
